# Kredi Kartı PDF Ekstre Analizi

## Kurulum

```bash
pip install -r requirements.txt
```

## Çalıştırma

```bash
streamlit run app.py
```

Sonra tarayıcıda açılan ekrana PDF ekstrelerini yükle.

## Özellikler

- Birden fazla PDF yükleme
- Günlük harcama analizi
- Şehir bazlı harcama analizi
- Kategori bazlı harcama analizi
- Mağaza bazlı analiz
- CSV dışa aktarma
- Yurt dışı harcamalarda EUR/USD bilgisini yakalama

## Not

Bu ilk sürüm Garanti Bonus PDF ekstre formatına göre hazırlandı.
Farklı banka PDF'lerinde regex kısmı güncellenebilir.
